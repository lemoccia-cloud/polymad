Como funciona

Três passos para transformar dados climáticos em oportunidades de retorno.

01

Escaneamos os mercados

Buscamos todos os mercados de temperatura no Polymarket via Gamma API e extraímos os preços implícitos do mercado.

02

Calculamos o edge real

Comparamos a probabilidade do modelo ECMWF IFS (51 membros ensemble) com o preço do mercado. A diferença é o edge — seu lucro esperado.

03

Você vê o retorno

Para cada oportunidade, mostramos quanto apostar (Kelly fraction), quanto ganhar se acertar e o retorno percentual esperado.

Tudo que você precisa

Modelos ensemble de 51 e 31 membros. As melhores previsões meteorológicas disponíveis no mundo.
